
# Trip Planner Agent


### Commands to run for UV 

`uv init <your-project-name>`

## Create virtual env 
`uv venv <your-env-name> -p <your-python-version>`

## Check available env
`uv python list`

## Install specific version with uv
`uv python install <ypur-python-version>`

## Check the dependency list
`uv pip list`
